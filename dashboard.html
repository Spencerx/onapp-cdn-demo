Video Embed Instructions

Replace "mystream" with the streamname

Our easy video embed script automatically detects the browser type (Desktop or Mobile device) and loads the appropriate player. Currently, this is either Flow Player or the browser’s native HTML5 player. The streaming protocol is also set appropriately.

To use it, enter into your webpage:

<html>
  <head>
    <script src="http://video.worldcdn.net/player.js" type="text/javascript"></script>
  </head>
  <body>
    <div id="my-video-player" height="640" width="360" />
    <script type="text/javascript">
      CDNPlayer("my-video-player",RESOURCEID, "mystream" );
    </script>
  </body>
</html>

Publishing instructions

Alternatively you can use the following script to generate the appropriate URL for publishing to your player:

<script>
function MyCallBack(urls) {
  // load the flash / html5 player with one of the urls
}
</script>
<script src="http://video.worldcdn.net/RESOURCEID/mystream.jsonp?callback=MyCallBack" />

Internal Live Streaming CDN Resource

Streaming instructions

NOTE: Required Publishing Point Settings (to be able to retrieve with Silverlight Player):

  • h.264 Baseline 3.
  • Audio should be AAC or MP3-stereo-44100Hz.
  • Key frame frequency should be 2 seconds.
  • Lower bitrate.

  1. Install Adobe Live Encoder
  2. Fill in the form:
    FMS URL: rtmp://.publishstream.worldcdn.net/P
    Backup URL: rtmp://backup..publishstream.worldcdn.net/P
    Stream: Your stream name
  3. Click "Connect"
  4. In the password popup, enter "P" as the username and "" as the password.
  5. Click "Start" to start publishing the live stream.
